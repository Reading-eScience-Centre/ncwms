SLD Parser Test Tool

Select one of the example XML files below, alter it if desired in the text area or paste your own example. XML that is compliant with Styled Layer Descriptor (SLD)/Symbology Encoding (SE) and uses the RasterSymbolizer tag will be accepted along the lines of the examples "SE Example", "Thresholds 1", "Thresholds 2" and "Interpolation". Press submit and please wait for a map of the globe to be generated on the right using one of the variables available in the datasets listed below. To refer to a variable use the machine readable ID in brackets. In most cases this should be quoted in the name of the styled layer except where two variables are referenced.
